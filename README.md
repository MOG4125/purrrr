# What is purrrr
purrrr is an installer for all Tabby Cat mods (Tabby Cat the chrome extension, not a mod to an actual Tabby Cat!)

# How to install purrrr?
just download the purrrr.py file and run it
<img width="752" height="468" alt="image" src="https://github.com/user-attachments/assets/360f305b-fc10-459a-a05f-21d335cf9a3d" />

It will bring up your file explorer, select the Tabby Cat mod file you want. P.S. it will look a little blurry/outdated
