# What is purrrr
purrrr is an installer for all Tabby Cat mods (Tabby Cat the chrome extension, not a mod to an actual Tabby Cat!)

# How to install purrrr?
Just download the purrrr.py file and run it
<img width="752" height="468" alt="image" src="https://github.com/user-attachments/assets/360f305b-fc10-459a-a05f-21d335cf9a3d" />

It will bring up your file explorer, select the Tabby Cat mod file you want. P.S. it will look a little blurry/outdated
<img width="938" height="709" alt="image" src="https://github.com/user-attachments/assets/49cb3fce-cd5f-4543-aae0-5df131f3af81" />

It will ask this:
<img width="752" height="468" alt="image" src="https://github.com/user-attachments/assets/ca058701-8f1f-46ff-9dff-4fa2b16ccfa4" />

If you press "Yes" it will wait for you to select your font file (for mods like Ferris) It accepts ttf, otf, woff and woff2 files.

If you press "No" this will come up:
<img width="938" height="709" alt="image" src="https://github.com/user-attachments/assets/97aa748c-8ec6-4882-9a17-d6cc3322c5e7" />

Just press "Select Folder" without selecting anything, once you have pressed that button, this will pop up:
<img width="752" height="468" alt="image" src="https://github.com/user-attachments/assets/bf101001-4679-44b4-9806-9cb91d1bcabf" />

Do what it says, this will also pop up even if you also select a font.


If you have made a new (or old) Tabby Cat hack that is not on this installer just email me at caiiummog@gail.com (:
